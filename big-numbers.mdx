# Big Numbers

Prominently display a single value in a block.

### Big Number Text Content

The **Big Numbers** component displays a single number value in a large font and two smaller, optional text values. The Big Numbers component is ideal for showing top-level reporting numbers. You can combine it with the [Chart Component](/charts) and [Data Grid Collection](/data-grid-collection) to create useful dashboards in your apps.

Big Number components can also be configured with actions. Adding an action to a Big Number component allows users to tap or click the component to trigger and action or workflow. 

## Configure a Big Numbers component



### Big Numbers - Arcade

### Big Numbers Block 2

1. Locate the **General** tab on the right side of the [Layout Editor](/intro-to-layout). 
2. Use the **Value** field to select the value from your data to display, or select **Custom** to enter a [Custom Value](/custom-values) for this field. This value will be the big number. The value can be any column from your table; a [Basic Number Column](/basic-columns), [Rollup](/rollup-column), [Lookup](/lookup-column), [Math Column](/math-column), or anything else.
3. The other two values for **Name** and **Description** are, by default, custom values that you type in directly while configuring the component. However, you can also make these dynamic and link them to a column in your table.

You can add multiple Big Number blocks inside of a single Big Number component. Just click **Add item** to add a new Big Number to your list and click and drag on the three dots in the top left to rearrange.
