### Link Body Text

You can add a text link that points to a destination within your app, a file to download, or to an external website.

1. In the [Layout Editor](http://intro-to-layout/), select the tab you want to edit.
2. Click on the plus (+) symbol in the [Components](http://components/) panel.
3. Select the **Links** component.
4. Locate the **General** tab on the right side of the Layout Editor. Use the **Link to** field to select a URL column from your data to display, or select **Custom** to enter a [Custom Value](/custom-values) for this field.
5. Configure the **Style** of the link to be displayed.- **Compact** style will display the link as a button with a width that adjusts depending on the chosen **Title.**
- **Row** will display the link in a full-width row along with an optional **Caption** field.
6. Select the title. You can either select a column with pre-populated value from your [Data Source](/data-sources) or add custom text by selecting **Custom**. The **Title** can also be left blank if you only want to display an Icon.
7. Select an **Icon** to be displayed.
8. If using the **Row** style, use the **Caption** field to select the value from your data to display or select **Custom** to enter a custom value for this field.
