# Rich Text

Display data as styled content.

### Rich Text Body 1

With the **Rich Text** component, you can apply italic text, bold text, different text sizes, inline images, links, and more using Markdown syntax.

## Adding and Setting up a Rich Text Component



### Adding Rich Text Arcade

### Rich Text Body 2

1. In the [Layout Editor](http://intro-to-layout/), select the tab you want to edit.
2. Click on the plus (+) symbol in the [Components](http://components/) panel.
3. Find **Text **in the menu and select **Rich Text**.
4. In the **General** tab on the right side of the Layout Editor, select the column you want to display or enter a [Custom Value](/custom-values). If you select a column from your data source that contains Markdown, it will be displayed with the specified styling in your app. You can also apply Markdown styling to custom text.

## Actions



### Rich Text Component Actions

### Rich Text Body 3

You can set up specific [Actions](/getting-started/introduction-to-actions) that trigger when users click on the Rich Text component. You can choose from any of the actions available in Glide, or you can create a new action.

1. Open the **Action **menu on the right side panel. 
2. In **Configure Action**, select the dropdown menu to view action options.
3. Select the action to be triggered when users tap your Rich Text component.
4. If prompted, apply any additional settings in **Configure Action.**

## **Using Markdown**

If you select a column from your data source that contains Markdown, it will be displayed with the specified styling in your app. You can also apply Markdown styling to custom text. For an in-depth explanation of what Markdown is and how to use it, check out the [Markdown Guide](https://www.markdownguide.org/getting-started).

If you'd like to try a browser-based Markdown text editor, try [StackEdit](https://stackedit.io/).
