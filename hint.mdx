## Adding and Setting Up a Hint Component



### Hint Arcade

### Hint Body Text 2

1. In the [Layout Editor](/intro-to-layout), select the tab you want to edit.
2. Click on the plus (+) symbol in the** **[Components](/components) panel.
3. Find **Text** in the menu and select **Hint**.
4. The **Description **field is for the text of the hint. You can enter this manually or use a text column from your data as the source. 
5. Select a mood for the hint. 
6. Select an icon to display on the hint. 

## Adding Actions and Workflows to a Hint

You can set an [Action](/getting-started/introduction-to-actions) or [Workflow](/automation/workflows) to trigger when users click on the button inside the Hint component. You can choose from any of the actions available in Glide, or you can create a workflow.
