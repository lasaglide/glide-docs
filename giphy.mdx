# Giphy

Search for and retrieve GIFs using the Giphy API

### Giphy - Block 1

[Giphy](https://giphy.com/) is a dynamic online platform that offers an extensive library of GIFs, enabling users to discover and share animated images for a variety of purposes. By incorporating the **Get GIF** integration in Glide, you can effortlessly search for and add GIFs to your app, enhancing user experience and promoting engaging visual content.

### Adding Giphy Arcade

### Giphy - Block 2

1. In Glide, click the Settings icon in the upper-right corner.
2. Navigate to the **Integrations** tab and then select **Giphy**.
3. Click the **Add **button.
4. Next, log into the [Giphy Developer Dashboard](https://developers.giphy.com/dashboard/).
5. Select **Create an App, **select **API**, and complete the creation of your API key.
6. Back in Glide, enter this **API Key **in your Giphy Integration settings.

## Get Gif

The Giphy integration in Glide has one feature—**Get GIF.** This can be added to a [Component](/components) that supports [Actions](/actions) (e.g., a [Button component](/button-components)), or added to an action sequence in the [Workflow Editor](/getting-started/introduction-to-workflows#the-workflow-editor).
