# Copy to Clipboard

Copy a value to the user's clipboard.

### Copy to Clipboard Body 1

The **Copy to Clipboard** action copies a value to the user's clipboard.



![Copy to Clipboard Hero](https://images.ctfassets.net/yreygbm72t08/6PuHrfszwOTjgDOcwDNw9A/98ddc41c1cabff6e42518786e92e2896/CleanShot_2024-10-03_at_5_.51.09_2x.png)



The **Copy to Clipboard** action can be used as an action in the components of your app (like [Buttons](/essentials/components/button), [Collections](/essentials/components/collection), and much more), or it can be triggered along with other actions as part of a [Workflow.](/getting-started/introduction-to-workflows)

## Setting up the Copy to Clipboard Action

### In the Layout Editor



### Copy Clipboard Layout Arcade

### Copy Clipboard Body 2

1. Click on the plus (+) sign in the **Components** panel and select the **Component** you want to trigger the action.
2. In the **General** tab on the right side of the Layout Editor, navigate to the **Actions** section and select **Copy to clipboard**.
3. Select the icon and descriptive text to use in your app.
4. Set which value the user can copy. You can add custom text, or one of several items from your data:- Current date/time
- Unique Identifier
- User's email address
- Link to current screen
- You can also select a column from your **data source**, or a column from the profile of the user who is logged in.
5. You can optionally add **Conditions** for certain criteria to be met before a user can trigger this action.

### In the Workflow Editor



### Copy Clipboard Action Editor Arcade

### Copy Clipboard Body 3

1. Select the **+ New Workflow** button. If you have existing workflows, click on the plus (+) sign in the left panel.
2. Select the **data source** for your action.
3. Configure your first action by clicking the first step in the sequence. You can also add conditions to your actions, if needed.
4. To add new actions to the flow, hover just underneath an action and click the plus button. In this way you can add the Copy to clipboard action as just one step in a larger workflow.
