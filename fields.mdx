# Fields

Show multiple values side-by-side.

### Fields Body Text 1

The **Fields** component allows you to place data in a table format.

## Setting up Fields



### Fields Arcade

### Fields Body Text 2

To set up your Field component:

1. In the [Layout Editor](/intro-to-layout), select the tab you want to edit.
2. Click on the plus (+)** **symbol in the [Components](/components) panel.
3. Select the **Fields** component.
4. Locate the **General** tab on the right side of the **Layout Editor**. 
5. Use the menus in the Options section to select the values from your [Data Source](/data-sources/) to display as the **Label** and **Value** for your field, or select **Custom** to enter [Custom Values](/custom-values).
6. You can add multiple Fields blocks inside a single Fields component. Just click **Add item** to add a new Field to your list and click and drag on the three dots next to each field to rearrange.
