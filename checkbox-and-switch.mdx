# Checkbox and Switch

Change true-false values.

### Checkbox + Switch Body Text

The **Checkbox** and **Switch** components allow you to toggle boolean values in your data source.

## Adding and Setting Up the Checkbox and Switch Components



### Checkbox and Switch Arcade

### Checkbox + Switch Body 2

1. In the [Layout Editor](/intro-to-layout), select the tab you want to edit.
2. Click on the plus (+) symbol in the **Components **panel.
3. Select **Checkbox **or **Switch**. 
4. Select a column in your [Data Source](/data-sources) to write the value to.
5. Use the **Label** and **Description** fields to let users know what the switch or checkbox is for.

If you like, you can change your component from Checkbox to Switch (and vice versa) in the configuration panel on the right side of the Layout Editor.
