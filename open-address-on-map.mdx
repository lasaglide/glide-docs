# Open Address on Map

Trigger the device's default map application.

### Show Address on Map Body Text 1

The **Open Address on Map** action allows users to open an address in their default map application on a mobile device, and in their default browser on a desktop device.



![Show Address on Map Hero](https://images.ctfassets.net/yreygbm72t08/6jrQbQaI65YFHPmJmpm8q0/38bf49d650c5b67c8b02f9ffabf0d1bd/CleanShot_2024-10-04_at_10_.29.41_2x.png)



The **Open Address on Map** action can be used as an action in the components of your app (like [Buttons](/essentials/components/button), [Collections](/essentials/components/collection), and much more), or it can be triggered along with other actions as part of a [Workflow.](/getting-started/introduction-to-workflows)

## **Setting up the** Open Address on Map Action

### In the Layout Editor




### Open Address on Map Layout Arcade

### Show Address on Map Body Text 2

1. Click on the plus (+) sign in the **Components** panel and select the **Component** you want to trigger the action.
2. In the **General** tab on the right side of the Layout Editor, navigate to the **Actions** section and select **Open address on map**.
3. Select the icon and descriptive text to use in your app.
4. In the **Address field**, select the address you want to user to visit. You can select a column in your data that has the address or manually enter an address. You can also use the latitude and longitude instead of a street address.
5. You can optionally add **Conditions** for certain criteria to be met before a user can trigger this action.

### In the Workflow Editor



### Show Address on Map Action Editor Arcade

### Show Address on Map Body Text 3

1. Select the **+ New Workflow** button. If you have existing actions, click on the plus (+) sign in the left panel.
2. Select the **data source** for your action.
3. Configure your first action by clicking the first step in the workflow. You can also add conditions to your actions, if needed.
4. To add new actions to the flow, hover just underneath an action and click the plus button. In this way you can add the Show Address on Map action as just one step in a larger workflow.
